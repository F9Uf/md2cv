|-----------|----------|-----------|-------------|-----------------|

| T-12-10 | Cross-Site Scripting | repo/file names rendered as JSX text | mitigate | All API strings are rendered as React text children (auto-escaped), not via `dangerouslySetInnerHTML`; no interpolation into HTML |
| T-12-11 | Tampering | empty/whitespace commit message | mitigate | Commit button disabled until `message.trim()` is non-empty, preventing empty-message commits |
