   ```ts
   for (const t of listTokens) {

       currentEntry.details.push(t.content)
     }
   }
   ```

   ```ts
   for (const t of listTokens) {

     }
   }
   ```
   `md.renderInline()` converts inline markdown tokens (bold, italic, code, links) to HTML and, with `html: true`, passes through inline HTML tags unchanged.
