<objective>
Author the on-screen page-chrome stylesheet that paints paged.js's emitted `.pagedjs_page` rectangles as white A4 sheets on a gray surround with a subtle Word/Docs-style drop shadow, a 16px vertical gap between stacked pages, and a 1px border. PREV-01 (visible page rectangle) and PREV-02 (visible second page rectangle on overflow) both require this chrome to be visible — without it the preview just looks like a slightly differently-margined block of text.

Output: A new `src/styles/pages.css` (created from scratch — no in-repo content analog) imported from `src/main.tsx`. No changes to `src/index.css`, `src/styles/themes.css`, or any `src/styles/theme-*.css`.
