**New TemplateClasses interface** (required keys per D-05):
```typescript
export interface TemplateClasses {
  container: string      // outermost wrapper div
  h1: string             // candidate name / first h1
  h2: string             // section heading
  h3: string             // entry title
  p: string              // paragraph
  ul: string             // unordered list wrapper
  ol: string             // ordered list wrapper
  li: string             // list item
  code: string           // inline code
  pre: string            // fenced code block wrapper
  a: string              // hyperlink
  blockquote: string     // blockquote
  hr: string             // horizontal rule
}
```

`TemplateName` type stays unchanged: `'classic' | 'modern' | 'minimal'`

**Mapping from old semantic keys to new element keys** (preserving visual output per D-07):
- `container` → `container` (unchanged)
- `name` → `h1`
- `sectionHeading` → `h2`
- `entryTitle` → `h3`
- `entryDetail` → `li`
- `detailList` → `ul`
- `preamble` / `extra` → `p` (use preamble value; extra had similar intent)
