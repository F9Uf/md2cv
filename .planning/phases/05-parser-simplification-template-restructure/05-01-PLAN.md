The new test file has two describe blocks:

**Block 1: "parseResume — Phase 5: HTML string output"**

All tests assert that `parseResume(markdown)` returns a string. Keep the same test scenarios from the old file but change assertions from `result.name`, `result.sections[0].entries[0].details[0]` etc. to checking the returned HTML string directly.

New assertions (exact patterns):

1. Returns a string type:
```
const result = parseResume('# Jane Doe\n')
expect(typeof result).toBe('string')
```

2. h1 produces `<h1>` element:
```
const result = parseResume('# Jane Doe\n')
expect(result).toContain('<h1>')
expect(result).toContain('Jane Doe')
```

3. bold → `<strong>` (STYLE-01):
```
const result = parseResume('**bold text**\n')
expect(result).toContain('<strong>bold text</strong>')
```

4. italic → `<em>` (STYLE-02):
```
const result = parseResume('*italic text*\n')
expect(result).toContain('<em>italic text</em>')
```

5. inline code → `<code>` (STYLE-03):
```
const result = parseResume('`inline code`\n')
expect(result).toContain('<code>inline code</code>')
```

6. link → `<a href>` (STYLE-04):
```
const result = parseResume('[click here](https://example.com)\n')
expect(result).toContain('href="https://example.com"')
expect(result).toContain('click here')
```

7. inline HTML not escaped (HTML-01 / PARSER-02):
```
const result = parseResume('<span style="color:red">x</span>\n')
expect(result).toContain('<span style="color:red">x</span>')
expect(result).not.toContain('&lt;span')
```

8. h2 produces `<h2>`:
```
const result = parseResume('## Experience\n')
expect(result).toContain('<h2>')
expect(result).toContain('Experience')
```

9. h3 produces `<h3>`:
```
const result = parseResume('### Job A\n')
expect(result).toContain('<h3>')
expect(result).toContain('Job A')
```

10. bullet list produces `<ul>` and `<li>`:
```
const result = parseResume('- item one\n- item two\n')
expect(result).toContain('<ul>')
expect(result).toContain('<li>')
expect(result).toContain('item one')
```

11. empty string → returns string (may be ''):
```
const result = parseResume('')
expect(typeof result).toBe('string')
```
